# Market Scan — Skill Demand Intelligence

What the market actually wants, based on real JD analysis.
Run this once, update monthly.

## How to run this scan

1. Pull 20–30 JDs from target roles (use LinkedIn/Naukri/jobs.ch saved searches)
2. Paste each JD into gap-analysis/jobs/ using the template
3. Tag every requirement with one of the skill buckets below
4. Tally frequency → fill the table below
