# YouTube — Strategy & Video Tracker

## Channel strategy

**Positioning**: Engineer who connects hardware and AI. Tutorials that no one else makes.
**Format**: Technical tutorials + career explainers. 15–40 min. Code on screen.
**Upload frequency**: 1×/month to start. Quality over frequency.
**Niche**: The hardware × AI intersection — zero competition, real audience.

## SEO approach

YouTube is a search engine. Titles must be searchable:
- ✓ "Building a Neural Network Accelerator in Verilog (from scratch)"
- ✗ "My cool hardware AI project"

Description: first 3 lines must be the pitch. Include GitHub link in first 3 lines.
Tags: 10–15 relevant tags (match what people search).

## Video production template

### Pre-production (1 hr)
- [ ] Script outline (not word-for-word — use bullet points per section)
- [ ] Code written and tested end-to-end
- [ ] GitHub repo README done
- [ ] Thumbnail concept sketched

### Recording (1–2 hrs)
- [ ] Record in one session (fix in post if needed)
- [ ] Show terminal + code side by side
- [ ] Narrate what you're thinking, not just what you're doing
- [ ] Record a 30-second intro separately (higher energy)

### Post-production (1 hr)
- [ ] Cut dead air and mistakes
- [ ] Add chapter markers (every major section)
- [ ] Add captions (auto-generated, review for accuracy)
- [ ] Export: 1080p minimum

### Upload checklist
- [ ] Title: searchable + specific (include main keyword)
- [ ] Thumbnail: face + text + high contrast; legible at 120px wide
- [ ] Description: GitHub link + summary in first 3 lines + timestamps
- [ ] Tags: 10–15 (main topic + adjacent topics)
- [ ] First comment: pinned with chapters + GitHub link
- [ ] LinkedIn post announcing it (30 min after upload)
