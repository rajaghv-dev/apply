# Lesson Plans

Each lesson plan is a self-contained learning module:
a defined scope, resources, exercises, an evidence artifact, and a done criterion.

When you complete a plan:
1. Mark it `STATUS: DONE` in the file
2. Add the artifact link
3. Update `profile/my-profile.yaml` with the new skill level + evidence
4. Add an entry to `second-brain/learning-log.md`
5. Stub a content piece in `content/ideas.md` if there's a post angle
