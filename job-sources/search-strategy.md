# Search Strategy

How to run systematic searches without burning hours on noise.

## Weekly search routine

### Tier 1 — check every week (15 min)
These move fast or are most relevant to the target profile:

1. **LinkedIn Jobs** — saved searches for each role type + location
2. **Naukri** (if targeting India) — set job alerts
3. **jobs.ch** (if targeting Switzerland) — set alerts
4. **Company career pages** (target list in company-careers.md) — check directly for roles not posted on aggregators

### Tier 2 — check every 2 weeks (30 min)
2. Indeed / Glassdoor for salary benchmarking
3. Honeypot / Otta / Landing.jobs for EU tech
4. IEEE Jobs / SemiEngineering / ChipDesignJobs for hardware roles
5. eFinancialCareers for finance-AI roles
6. LegalGeek Jobs for legal tech

### Tier 3 — monthly (research, not applying)
- EURES for EU mobility context
- XING for DACH network
- iimjobs for India senior roles

## LinkedIn Saved Searches (set these up)

Run these and save as LinkedIn alerts:

### Generalist / Staff Engineer

## Outreach Template (cold DM to recruiter)

Subject: [Role Title] — [Your name] | [2-line differentiator]

```
Hi [Name],

I noticed you're hiring for [role] at [company]. I'm a technologist with end-to-end 
experience from chip design to full-stack apps, with recent focus on [AI/LLM / legal tech / 
fintech] — which aligns well with what I see in this role.

Happy to share more. Would a 20-min call work?

[Your name]
```

Keep it under 5 lines. Attach nothing. Follow up once after 5 days.
